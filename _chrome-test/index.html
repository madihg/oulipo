This page exists only to verify the persistent UI chrome works standalone before we integrate it into every site page. Nothing else.

What you should see

  • Thin top signup bar with halim madi on the left, an email field, and a subscribe button.
  • Top-right: a small connect [c] chip.
  • Bottom-right: human [h] / machine [m] pill (HUMAN active by default).

Keyboard shortcuts

  • c goes to /connect/.
  • h exits MACHINE mode (closes palette).
  • m enters MACHINE mode (opens palette).
  • / opens palette without changing mode.
  • ⌘K / Ctrl+K opens palette from anywhere (incl. inputs).
  • ↑ ↓ navigate palette suggestions; Enter fires; Esc closes; Tab autocompletes.
  • When the palette is open, /+⌘K hint chips appear above the bottom-right pill.

Try the palette

Press m. You should see a dark overlay with a monospace site map dimmed at low opacity, a list of slash commands above the input bar, and a ›-prompt input at the bottom. Type works or book or connect and the list filters live.

Try the signup form

Type any email into the top bar and submit — you should see subscribing… then either subscribed ✓ (best-effort) or a try at substack.com ↗ fallback link. Real Substack wiring lands in US-006.
