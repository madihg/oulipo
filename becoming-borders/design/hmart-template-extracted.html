01

principles

seven rules that distinguish this system from a generic editorial site. if a design choice doesn't trace back to one of these, it probably doesn't belong.

01 / hierarchy through opacity
black on white, stepped down by opacity.

no off-whites, no tinted grays, no shadows. five levels of black-alpha do all the heavy lifting that color usually does.

02 / color is identity, not chrome
accent color belongs to a performance. it never paints the page.

the five accents are functional moments - top border, status pill, focus ring, button, cursor. white is the only background.

03 / type carries the voice
distressed display + swiss variable sans + mono captions.

the contrast between a humanist face and a clean variable family is the signature move. one display moment per page, used big.

04 / white space replaces ornament
negative space is the decoration.

a tighter reading column, generous vertical rhythm, hairline rules. nothing else needs to assert presence.

05 / interactions are short, opacity-led
hover drops to 0.7. selection drops siblings to 0.5.

no bouncing, no sliding. the language is opacity and grayscale-to-color, both 0.3 - 0.5s.

06 / poetry is the content the chrome serves
poem text uses pre-line. line breaks and stanzas are sacred.

never reflow whitespace inside a poem. voting is blind during training - no labels reveal author.

07 / a print catalogue, not a marketing site
reads like a zine. lower-case wordmark, hyphens not em dashes, no emoji.

declarative copy, comma-separated taxonomies, first-person where it appears. visual ornament is replaced with white space.

02

voice

direct, image-led, anti-marketing. how copy reads matters as much as how the chrome looks.

do

  • lower-case wordmarks (singulars, not Singulars)
  • declarative sentences. short. image-led.
  • comma-separated taxonomies (non-human poets, anti-gravitational word interfaces, somatic semantics)
  • first-person where it appears
  • hyphens with spaces ( - ) for parentheticals
  • "kitchen laboratory" framing - humble, tactile, hand-made

don't

  • no em dashes, ever
  • no emojis
  • no marketing puffery ("revolutionary", "seamless", "delightful")
  • no clever icon set - the only glyph is ↗
  • no title case in chrome
  • no exclamation points

03 — uniform opacity hover

08

accessibility

built into the system, not bolted on. accent color is the place where it gets enforced programmatically.

  • WCAG AA enforced on accent text via ensureContrast() / accessibleTextColor(). binary search darkens the hue until 4.5:1 is met. raw hex never lands on a glyph.
  • UI accent on borders darkened to 3:1 via accessibleUIColor().
  • Focus rings always visible. 3px solid, 2px offset, in accent. never outline: none.
  • Custom cursor is hover affordance only. default pointer is the real interactivity signal.
  • Poem structure preserved. white-space: pre-line keeps line breaks for screen readers.
  • Voting is blind during training. no human/machine labels on cards. the model is judged on the work.
